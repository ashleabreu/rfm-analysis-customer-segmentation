# rfm-analysis-customer-segmentation
## Descripción del proyecto
Este proyecto tiene como objetivo segmentar clientes utilizando el modelo RFM(Recency, Frequency, Monetary) con el fin de identificar clientes de alto valor, clientes en riesgo y oportunidades de retención, apoyando la toma de decisiones de negocio.

---
## Contexto del problema
Las empresas necesitan entender el comportamiento de sus clientes para optimizar estrategias de marketing y retención.
El ánalisis RFM permite clasificar a los clientes en función de:
- **Recency**: qué tan reciente fue la última compra
- **Frequency**: con qué frecuencia compra el cliente
- **Monetary**: cuánto dinero ha gastado el cliente

---
## Herramientas utilizadas
-**SQL**: cálculo de métricas RFM y segmentación de clientes
-**Power BI**: visualización y ánalisis de resultados
-**Excel / Power Query**: limpieza inicial de datos 

---
## Proceso de ánalisis
1. Limpieza y preparación de datos
2. Cálculo de métricas RFM mediante consultas SQL
3. Asignación de puntuaciones RFM a cada cliente
4. Segmentación de clientes según su comportamiento
5. Visualización de los segmentos en Power BI

---
## Resultados principales
- Identificación de clientes de alto valor (VIP)
- Detección de clientes con riesgo de abandono
- Segmentos con mayor contribución monetaria
- Insights útiles para estrategias de retención

---
## Aprendizajes 
- Uso de consultas SQL más avanzadas para ánalisis de negocio
- Aplicación práctica del modelo RFM
- Interpretación de datos para apoyar decisiones empresariales
- Integración de SQL con Power BI para ánalisis visual
